# 시지각 확장 훈련

## Visual Span

한 눈에 읽는 글자의 수\(visual span\)을 확장하기 위한 훈련입니다.

```text
$ give me super-powers
```

{% hint style="info" %} Super-powers are granted randomly so please submit an issue if you're not happy with yours. {% endhint %}
